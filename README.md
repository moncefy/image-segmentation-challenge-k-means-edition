## Required Libraries
- numpy
- opencv-python
- matplotlib
- scikit-learn

You can copy the following command and paste it in your terminal to install all required libraries:
```
pip install numpy opencv-python matplotlib scikit-learn
```

## How to Run
1. Place your images in the `Images` folder (JPG/PNG format).
2. Open a terminal in this project directory.
3. Execute:
```
python segment_images.py
```
4. Segmented images will be saved in the `outputs` folder.
